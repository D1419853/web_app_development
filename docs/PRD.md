# PRD：線上算命系統

## 1. 專案概述
- **背景與動機**：現代人生活與工作壓力大，常面臨職涯、感情或健康等抉擇。雖然傳統信仰提供了心靈慰藉，但實體宮廟或算命館不見得能隨時前往。本系統希望打造一個線上平台，讓使用者能不受時間空間限制，隨時進行算命、占卜與抽籤，並提供隨喜捐獻香油錢的功能，獲得心靈上的平靜與指引。
- **目標用戶**：
  - 面臨人生抉擇、需要快速獲得心靈指引的人。
  - 習慣在實體廟宇拜拜求籤，但因忙碌無法親自前往的信眾。
  - 對命理、占卜或塔羅感興趣，並喜歡在社群上分享運勢的年輕族群。
- **核心價值主張**：兼具傳統儀式感與現代便利性的線上算命平台，整合儲存紀錄與線上香油錢捐款，隨時隨地提供專屬的心靈寄託。

## 2. 功能需求
本系統主要功能包含以下五項：

1. **線上算命、占卜、抽籤**
   - 作為 **使用者**，我希望 **能夠在線上流暢地進行抽籤與算命**，以便 **在遇到困難或疑惑時獲得指引**。
2. **儲存算命結果**
   - 作為 **使用者**，我希望 **系統能記錄我過去的算命與抽籤結果**，以便 **我日後隨時回顧與驗證當時的結果**。
3. **捐香油錢**
   - 作為 **信眾**，我希望 **能直接透過線上支付捐獻香油錢**，以便 **還願或表達支持、祈求好運**。
4. **會員註冊與登入**
   - 作為 **使用者**，我希望 **能建立專屬個人帳號**，以便 **安全地管理我的算命紀錄與個人資料**。
5. **社群分享運勢結果**
   - 作為 **使用者**，我希望 **能將有趣的抽籤/算命結果一鍵分享到社群平台**，以便 **與親朋好友交流或分享今日好運**。

## 3. 非功能需求
- **技術限制**：
  - 後端框架：Flask
  - 模板引擎：Jinja2
  - 資料庫：SQLite
- **效能與安全考量**：
  - 確保捐獻香油錢的頁面或模擬交易流程穩定且具備防呆機制。
  - 會員密碼須以雜湊函數（例如 Bcrypt）加密後存入資料庫以防資料外洩。
  - 前端抽籤的動畫（如擲筊、搖籤筒）需優化，以呈現流暢自然的使用者體驗，不應有嚴重卡頓。
  - 需要保護使用者隱私，個人的算命與占卜結果不得公開給未授權的訪客。

## 4. MVP 範圍
- **Must Have (必須有)**：
  - 基礎線上抽籤、解籤邏輯（如產生隨機籤詩）。
  - 香油錢捐獻介面（可先為顯示轉帳帳戶或模擬金流按鈕）。
  - 直觀順暢的基礎算命頁面。
- **Should Have (應該有)**：
  - 基礎的會員系統（註冊/登入）。
  - 儲存個人算命歷史紀錄的頁面。
- **Nice to Have (加分項)**：
  - 寫實的抽籤/擲筊動畫效果。
  - LINE/Facebook/IG 一鍵社群分享結果卡片的功能。
  - 多樣化的算命模組（例如除了求籤，還有塔羅牌抽牌）。
