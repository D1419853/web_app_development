# 線上占卜系統 — 產品需求文件 (PRD)

> **版本：** v1.0
> **建立日期：** 2026-04-09
> **專案名稱：** 線上占卜系統
> **目標用戶：** 一般大眾、尋求心靈指引或娛樂者

---

## 1. 專案概述

### 1.1 背景與動機
在忙碌的現代生活中，人們常感到迷惘或需要一點心靈寄託。傳統的抽籤、占卜與算命是常見的文化體驗，透過將其數位化，可以讓使用者不受時間、地點限制，隨時進行簡單的心靈對話與自我反思。

### 1.2 目標用戶
- **尋求指引者**：在面臨選擇或壓力時，希望獲得一點建議或隨機啟發的人。
- **民俗愛好者**：對傳統抽籤、命理感興趣的用戶。
- **趣味體驗者**：想要在日常生活中體驗占卜樂趣的人。

### 1.3 核心價值主張
提供一個畫面精美、儀式感足夠且操作簡單的線上占卜平台，結合抽籤與紀錄功能，讓占卜不只是單次的隨機結果，更能成為個人心靈成長的小幫手。

---

## 2. 功能需求

### 功能一：抽籤、占卜、算命
> **使用者故事：** 作為一位追求指引的使用者，我希望能在線上抽籤、算命，以便在面臨困惑時獲得一些參考意見。
- **功能細節：** 
  - 模擬抽籤流程（如：點擊籤筒）。
  - 隨機生成占卜結果與解籤。
  - 不同類型的占卜選擇（如：求職、感情、健康）。

### 功能二：儲存占卜與抽籤結果
> **使用者故事：** 作為一位常客，我希望系統能儲存我的抽籤紀錄，以便我日後回顧當時的心境與運勢。
- **功能細節：** 
  - 登入後，自動將每一次的占卜結果記錄到資料庫。
  - 支援手動標記或收藏特定的好籤。

### 功能三：捐香油錢 (虛擬募資/功德箱)
> **使用者故事：** 作為一位重視儀式感的使用者，我希望能在占卜後捐贈一點香油錢，以便仿效真實廟宇的體驗，增加心理寄託感。
- **功能細節：** 
  - 使用者可以自由輸入虛擬金額。
  - 捐贈後顯示感謝語或特殊視覺回饋（如：點燈動畫）。

### 功能四：會員註冊與登入
> **使用者故事：** 作為一位忠實用戶，我希望有一個帳號系統，以便在不同裝置上都能查看到我的占卜歷史。
- **功能細節：** 
  - 提供基礎的帳號註冊與登入功能。
  - 管理個人 profile（如暱稱、頭像）。

### 功能五：歷史紀錄查詢
> **使用者故事：** 作為一位使用者，我希望有一個專屬頁面查看我過去所有的占卜紀錄，以便對比我的運勢變化。
- **功能細節：** 
  - 列表式顯示過去的抽籤、算命結果。
  - 支援按時間或占卜類型篩選。

---

## 3. 非功能需求

### 3.1 技術限制
- **後端**：Flask (Python)
- **前端**：Jinja2 + CSS (強調視覺與動態效果)
- **資料庫**：SQLite (供本地存儲與輕量部署)

### 3.2 效能與安全
- **安全性**：使用者密碼必須經過雜湊處理。
- **響應式設計**：需支援行動裝置與桌面端 (RWD)。
- **使用者體驗**：占卜過程應包含適當的動畫，提升互動趣味。

## 4. MVP 範圍

### 🔴 Must Have (核心功能)
- 基礎抽籤與解籤系統
- 會員註冊、登入功能
- 占卜歷史紀錄儲存

### 🟡 Should Have (應有功能)
- 線上捐香油錢 (模擬)
- 歷史紀錄查詢與篩選

### 🟢 Nice to Have (加分項)
- 每日自動推送運勢
- 社群分享功能
