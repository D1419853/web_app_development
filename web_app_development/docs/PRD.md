# 線上占卜系統 — 產品需求文件 (PRD)

> **版本：** v1.0
> **建立日期：** 2026-04-09
> **專案名稱：** 線上占卜系統
> **目標用戶：** 各年齡層人群

---

## 1. 專案概述

### 1.1 背景與動機
在繁忙的現代生活中，人們常感到迷惘或需要一點心靈指引。傳統的占卜、抽籤與算命是華人文化中重要的慰藉方式。透過線上化的方式，可以讓使用者不受時間、地點限制，隨時進行心靈對話，並系統化地管理自己的占卜紀錄。

### 1.2 目標用戶
- **各年齡層人群**：無論是年輕人對運勢的好奇，或是中老年人對指引的需求。
- **民俗文化愛好者**：希望體驗傳統智慧與現代科技結合的人。
- **尋求心靈平靜者**：在面臨壓力或抉擇時，需要一點方向性建議的人。

### 1.3 核心價值主張
提供一個**畫面精美、操作簡單、內容豐富**的線上占卜平台。不僅提供多樣化的占卜方式，還能儲存歷史紀錄，並透過虛擬捐款（香油錢）模擬真實廟宇體驗，打造沉浸式的心靈空間。

---

## 2. 功能需求

### 功能一：線上占卜、抽籤、算命
> **使用者故事：** 作為一位追求指引的使用者，我希望能在線上抽籤、擲筊或進行命理分析，以便在日常生活中獲得小小的建議或安慰。
- **功能細節：**
  - 提供多種模式：模擬抽籤筒（抽籤）、擲筊動畫（占卜）、基本八字或姓名分析（算命）。
  - 隨機生成籤詩內容，並附帶白話說明的解籤。
  - 搭配精美視覺動畫，提升儀式感。

### 功能二：儲存抽籤與算命結果
> **使用者故事：** 作為一位常客，我希望系統能自動記錄我的占卜結果，以便我日後回顧，觀察運勢的走向。
- **功能細節：**
  - 登入狀態下，每次占卜結果會自動存入資料庫。
  - 提供歷史紀錄頁面，使用者可以按時間、類型篩選查詢。
  - 支援刪除不想要的紀錄。

### 功能三：線上捐香油錢 (虛擬功德箱)
> **使用者故事：** 作為一位虔誠的使用者，我希望能在占卜後捐贈一點香油錢，以便仿效傳統習俗，增加心理寄託感。
- **功能細節：**
  - 使用者可以選擇捐獻金額（虛擬積分或數額輸入）。
  - 捐贈後顯示動畫與致謝頁面。
  - 記錄捐款紀錄（非實體金流，僅作為模擬體驗）。

### 功能四：會員註冊及登入
> **使用者故事：** 作為一位想要長期使用的使用者，我希望有一個帳號系統，以便在不同裝置上同步我的資料。
- **功能細節：**
  - 支援帳號、密碼註冊。
  - 登入、登出功能。
  - 個人資料管理（如更換頭像或暱稱）。

### 功能五：每日運勢推薦
> **使用者故事：** 作為一位日常使用者，我希望每天開啟網頁都能看到當日的精選語錄或運勢提示，以便開啟美好的一天。
- **功能細節：**
  - 每天自動更新首頁的「今日指引」。
  - 根據使用者的生肖或星座提供簡單的開運建議。

## 3. 非功能需求

### 3.1 技術限制
- **後端**：Flask (Python 3.12+)
- **前端**：Jinja2 + CSS (Vanilla CSS, 追求高品質動效)
- **資料庫**：SQLite (輕量級、易於部署)
- **環境**：支援行動端 RWD 響應式佈局。

### 3.2 效能與安全
- **加密**：使用者密碼必須經過雜湊 (Hash) 儲存，保障安全。
- **流暢度**：動畫執行需流暢，首頁載入限制在 2 秒內。
- **輸入驗證**：防止 SQL 注入與 XSS 攻擊。

## 4. MVP 範圍

### 🔴 Must Have (核心功能)
- 基礎抽籤與解籤系統
- 會員註冊、登入功能
- 占卜歷史紀錄儲存
- 基礎 RWD 介面設計

### 🟡 Should Have (應有功能)
- 擲筊動畫效果
- 線上捐香油錢 (模擬)
- 分享結果到社群的功能

### 🟢 Nice to Have (加分項)
- 背景音樂與音效
- 更多樣化的算命系統（如紫微斗數簡版）
- 即時占卜師在線對話（模擬 AI 聊天室）

## 6. 頁面規劃 (Site Map)

- **首頁 (`/`)**: 品牌故事、每日運勢入口、功能導覽。
- **占卜區 (`/divination`)**: 抽籤、擲筊、算命的子母頁面。
- **個人紀錄 (`/history`)**: 列表式顯示過去的占卜結果。
- **香油錢頁面 (`/donate`)**: 虛擬捐獻操作區。
- **帳號管理 (`/login`, `/register`, `/profile`)**: 使用者身份認證。
